## Research & Projects

### Morphology and Classification for E+A Galaxies (AAS 246)
Continuation of work on **quantitative morphology for SDSS galaxies** and **E+A classification**. Recently focused on training a **Random Forest classifier** using **color data + morphology parameters**. :contentReference[oaicite:4]{index=4}  
Poster: https://aas246-aas.ipostersessions.com/Default.aspx?s=AAS_246_Gallery (search my name) :contentReference[oaicite:5]{index=5}

### Initial Morphology for 748 E+A Galaxies (Spring 2025 ASNY)
Built an automated morphology procedure for large samples using only **RA/Dec** and a **single-color SDSS frame FITS** file; applied to the **Goto et al. (2003)** E+A catalog. :contentReference[oaicite:6]{index=6}

### Identifying E+A Galaxies with Broadband ugrizy Magnitudes
Comparing post-starburst samples from the **Goto E+A catalog** with newer E+A data from **SDSS MaNGA**. Scraped **SDSS DR7** to get colors/magnitudes for **748** post-starburst galaxies; recreated analysis from “Refining the E+A galaxy.” :contentReference[oaicite:7]{index=7}

### Per-Spaxel Star Formation Rate in SDSS MaNGA
Wrote a Python workflow to pull **per-spaxel SFR** data for MaNGA galaxies; exploring use cases including “E+A+” candidates. :contentReference[oaicite:8]{index=8}  
Short note (PDF): https://jaycup.neocities.org/pullingsfr.pdf  
(“Star Formation Rate data from SDSS Marvin”, **Nov 2023**) :contentReference[oaicite:9]{index=9}
